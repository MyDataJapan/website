過去にPeatix経由でMyData Japan のイベントへ参加された方へのお知らせ

2020年11月17日、Peatix社から個人情報流出のお知らせが公表されました。

2020年10月16日〜17日にかけて発生した不正アクセスにより、Peatix利用者のお客様情報（氏名、メール、暗号化されたパスワードなど）が最大677万件不正に引き出されたというものです。（現在のところクレジットカード情報、住所及び電話番号が引き出された事実は確認されておりません）

過去、MyDataJapanでは、以下のセミナーでPeatixを用いました。

  • 2018/05/25(金) 10:00-18:30　MyData Japan 2018
  • 2018/10/31(水) 18:00-20:30　MyData 101 vol.1 〜入門編〜
  • 2018/12/14(金) 19:00-21:30　MyData LINKS Identity
  • 2019/02/25(月) 19:00-21:35　MyData LINKS Healthcare
  • 2019/05/15(水) 10:00-18:45　MyData Japan 2019
  • 2019/05/17(金) 18:30-21:00　MyData Global - 情報銀行の震源地欧州からのムーヴメント
  • 2019/10/09(水) 18:00-22:00　MyData LINKS “The Great Hack”

当該セミナーに参加登録された方には、peatix社からのお知らせをご一読の上、パスワードの変更を強くおすすめいたします。

ご心配、ご迷惑をおかけし、お詫び申し上げます。

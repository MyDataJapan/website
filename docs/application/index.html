入会申込について

MyDataの活動に賛同いただけるかたは、ぜひ MyData Japan の会員になって、ご支援お願いします。

入会フロー

ステップ１

MyData原則への賛同
MyData原則をご参照ください。

ステップ２

入会申込
会員種別を選択し、入会申込をお願いいたします。

ステップ３

入会審査
理事会での入会審査が月一回行われます。

ステップ４

会費のお支払い
会費をお支払いいただき、会員となります！
